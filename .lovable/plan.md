

# Ultra-Premium Modular Furniture Website

## Overview
A cinematic, Apple/Tesla-inspired website for a modular furniture brand with smooth GSAP animations, parallax scrolling, and premium UX. The uploaded video will loop in the hero section.

## Design System
- **Colors**: Deep charcoal (#1a1a1a) primary, subtle gold (#c9a96e) accent, white/soft gray backgrounds
- **Typography**: Inter font family, large breathable spacing
- **Border radius**: 12-16px, soft and modern
- **Animations**: GSAP (ScrollTrigger, parallax, staggered reveals, horizontal pin scrolling)

## Sections

### 1. Navigation
- Minimal fixed navbar with logo, nav links, and "Get Quote" CTA
- Transparent on hero, transitions to solid on scroll

### 2. Hero Section (Split Layout)
- **Left**: Uploaded video playing in loop (muted, autoplay) with GSAP scale fade-in
- **Right**: Bold headline "Transform Spaces. Redefine Living." with text reveal animation
- Subtext + two CTA buttons ("Explore Designs" / "Get Quote")
- Subtle parallax on scroll

### 3. Product Transformation Section
- Full-width with scroll-triggered parallax layers
- Progressive text reveal: "From concept to reality — engineered transformations."
- Layered depth elements moving at different speeds

### 4. Features / Engineering Grid
- 4 feature cards: Precision Engineering, Modular Flexibility, Premium Materials, Smart Storage
- Icons with hover lift + subtle glow effect
- Staggered scroll-triggered entrance

### 5. Interactive Horizontal Showcase
- GSAP ScrollTrigger pinned section
- Panels slide horizontally as user scrolls vertically
- 4-5 showcase panels with images and text

### 6. Materials & Finishes
- Glassmorphism cards for Wood, Marble, Matte finishes
- Soft shadows, smooth fade transitions
- Luxury feel with subtle gradients

### 7. Before / After Transformation
- Interactive slider revealing before/after states
- Drag handle with smooth animation

### 8. Testimonials
- Auto-sliding minimal luxury cards
- Smooth fade transitions between testimonials

### 9. Call to Action
- Dark premium full-width section
- "Ready to transform your space?" + "Book Consultation" button
- Subtle background animation

### 10. Footer
- Minimal, elegant layout with links, social icons, copyright

## Technical
- Install GSAP + ScrollTrigger for all animations
- Copy uploaded video to src/assets for hero section
- Smooth 60fps performance throughout
- Responsive design for all screen sizes
